# SlowMo

>**Ever wanted to star in your own stick figure showdown? Gather a friend and fight them in a battle of wits! Who can outplay the other?**

## Operation Status
> This game is currently in a finished state, and I don't intend on adding more to it.

## Running and Building this Project
> Play this directly on my [itch.io](https://joshua-yoon1030.itch.io/66-68) page. Leave a like if you enjoy!

## Technologies Used And Motivations
> All game editing and creation was done in Godot 4.4, with all script files being written in GDScript.
> All art assets were created in Pivot Animator and imported into Godot.
> All game coding, and art assets were created by me.

> The two main aims of this project was to learn more about game design by creating my own [Game Design Document](https://docs.google.com/presentation/d/1n5G9fvhHUtCYFQOeQ5ZWE-V7Xu9asJ7nJCpCXDewu3M/edit?usp=sharing)
for this project before I began creating anything, and to learn more about animation.

> The premise of this game is simple, and therefore not very hard to implement. However, most of my time and struggles from this project came with the animations. I thought animating stick figures would be
a good starting point and simple enough to figure out on my own, but I starting from a completely new background and getting right into it made me deliberate over very simple things.
Overall, a very humbling and enjoyable experience.

> The Game Design Document was a good practice, and overall architecturing a project before I began made the process a lot smoother. When making this, I tried to follow the design principles outlined by [Masahiro Sakurai](https://www.youtube.com/watch?v=2Iiu1k16Ukc),
the director of the game Super Smash Bros. Brawl and someone who I study very carefully for game design advice. The actual content of this document is relatively simple, but the approach still made a big difference so I would consider my
experiment a success. In the future, to test my knowledge further I would like to document something much more complicated, but created by something else (for example a game mechanic in TeamFight Tactics, a game I am very familiar with).
